# Outreach-Funnel-Performance-Analysis-Conversion-Optimization
This project analyzes outreach and lead engagement data of an NGO Pehchaan to understand how different channels and profile types perform across the conversion funnel.  The objective was to measure response behavior, identify high-performing segments, and optimize outreach efforts using data-driven insights instead of manual tracking using **SQL**, **Power BI** & **MSExcel** 
Pehchaan is a Delhi based NGO focusing on Educating underprivileged children

## 🎯 Project Objective
The primary objective of this project is to:
-Outreach activities were being tracked manually with limited visibility into:
-Which channels generate the most replies
-Which profiles convert better
-How long leads take to respond
-Anticipate engagement trends
-Recommend actionable solutions



## 📂 Dataset
-552 total leads
-Profile types: Small Business, College Society, Restaurant, Bakery, Cafe, School, etc.
-Channels: Instagram, Email, LinkedIn, Phone, Website, Facebook
-Fields include contact status, reply status, response time, and outcomes

## ⚙️ Approach
## 1. Data Preparation
-Cleaned and standardized raw outreach data
-Created categorical fields (Channel, Profile Type, Reply Status)
-Calculated response time (days)

## 2. KPI Design
-Defined key funnel metrics:
-Total Leads
-Reply Rate
-Conversion Rate
-Positive vs Negative responses
-Channel effectiveness
-Average response time

## 3. Dashboard Development
-Built interactive dashboards to:
-Track funnel performance
-Compare channels and profiles
-Monitor response behavior
-Identify improvement opportunities

## 📈 Key Insights
-Total leads analyzed: 552
-Response rate: 6.16%
-Conversion rate: 2.36%
-Instagram was the most effective contact channel
-Some profile types showed significantly higher reply rates
-Average response time: ~5 days

## 🛠️ Tools Used
-Microsoft Power BI – Dashboarding & visualization
-Microsoft Excel – Data cleaning & preprocessing
-SQL – Structured querying
